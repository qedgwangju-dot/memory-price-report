# 메모리 가격 보고서 자동화

이 저장소는 `메모리 가격 보고서`를 매일 오전 6:30 KST에 GitHub Actions로 생성합니다.

최상위 고정 규칙: 최신성이 필요한 값은 모델 기억, 과거 대화, 이전 보고서, 검색 스니펫, 추정값, 보간값으로 채우지 않습니다. 반드시 현재 접근 가능한 공식/신뢰 소스의 웹/API/파일/DB를 직접 조회한 뒤 반영하고, 각 최신값에는 조회 시각, 출처, 원문 기준시각 또는 Last Update, 상태를 함께 표시합니다.

OpenAI API를 사용하지 않습니다. 스크립트가 직접 접근 가능한 웹/API만 조회하고, 조회 실패, 지연 데이터, 출처 간 불일치, 접근 제한, 기준값 부족이 있으면 값을 추정하거나 재사용하지 않고 `확인 불가` 또는 `지연/불일치 있음`으로 표시합니다.

이미지 요약판은 가장 엄격한 출력물로 처리합니다. 이미지 안에는 `확인` 상태의 직접 조회값만 숫자, 변화율, 판정 계산에 사용합니다. `확인 불가` 또는 `지연/불일치 있음` 상태인 값은 원본 객체에 숫자가 남아 있어도 이미지에서 숫자로 노출하지 않고 보류로 표시합니다. 출처·상태를 붙일 수 없는 하단 요약칩에는 숫자를 넣지 않습니다.

## 작동 방식

- 실행 시간: 매일 오전 6:30 KST
- GitHub Actions cron: `30 21 * * *` UTC
- 보고서 파일: `reports/memory_price_report_YYYY-MM-DD.md`
- 이미지 요약판: `reports/memory_price_summary_YYYY-MM-DD.png`
- 수동 실행: GitHub Actions 화면에서 `Memory price report` 워크플로우의 `Run workflow`

## 주요 직접 조회 소스

- USD/KRW: Yahoo Finance `USDKRW=X`
- DRAM/NAND/SSD 공개 가격: DRAMeXchange 홈페이지와 공개 `HomePrice` JSON
- 소매 가격: Danawa `prod.danawa.com` 가격비교 상품 페이지

## 기준

- 모델 기억, 과거 보고서, 검색 스니펫, 추정값, 보간값은 사용하지 않습니다.
- 같은 출처에서 직접 확인되지 않은 비교값은 계산하지 않습니다.
- 조회 실패, 지연 데이터, 출처 간 불일치, 접근 제한은 `확인 불가` 또는 `지연/불일치 있음`으로 남깁니다.
- OpenAI API 키나 GitHub Secret은 필요 없습니다.
